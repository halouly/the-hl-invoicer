# Invoicer — offline invoice generator

A standalone, fully offline invoice generator and tracker. The entire app is a single file — `hlearn-invoices.html` — with no build step, no server, and no dependencies. Open it in any modern browser and it runs.

## Usage
1. Download `hlearn-invoices.html` (or clone this repo).
2. Open it in your browser.
3. Set your business details under **Settings**, then start creating invoices.

Use **Export backup** / **Import backup** to move data between devices.

## Data & privacy
All data is stored locally in your browser's `localStorage`. No backend, no network calls. Export backups regularly.
